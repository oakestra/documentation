Why Oakestra?

Oakestra provides everything you need to run distributed applications reliably — from small edge devices to powerful cloud servers.

Edge-Cloud Native

Minimal overhead designed for constrained edge hardware, while staying fully compatible with the cloud-native ecosystem.

Efficient App Management

Deploy and control microservices via a powerful REST API, intuitive CLI, or a web dashboard — all from a single control plane.

Extensible by Design

Plug in custom scheduling algorithms, orchestration policies, addons, and hooks without touching the core.

Semantic Networking

Service-to-service communication with built-in semantic addressing — no manual IP management across heterogeneous nodes.

Multi-Runtime Support

Run containers, unikernels, and other virtualization technologies side-by-side on the same infrastructure.

Research-Backed

Born in academia and deployed in real-world edge testbeds. Featured at top-tier venues including IEEE, ACM, and USENIX.
